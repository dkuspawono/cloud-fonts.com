To use fonts from cloud-fonts.com, simply add the line 

<link href="http://www.cloud-fonts.com/format.php?font=FONTNAME" rel="stylesheet" type="text/css">

within the <head> </head> of your webpage

Text on your web pages can then be set with the font by using the font-family tag, like this;

font-family: 'FONTNAME';

If a bold version of the font is supported, just use the;

font-weight:bold;
rule in your css to set text with the bold version.
